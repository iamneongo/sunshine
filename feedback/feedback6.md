14. PROMPT CHO DEV GẮN CHATBOT VÀO LANDING

Build a real-estate sales chatbot for a NextJS landing page.

Goal:
- Convert visitors into leads
- Collect phone number / Zalo
- Qualify buyer intent
- Push to booking call / viewing schedule

Main headline context:
"Can ho dong tien chi tu 1,2 ty - tang gia ky vong 15%"

Bot behavior:
1. Greet user based on headline context
2. Offer 4 quick actions:
   - Nhan bang gia
   - Xem video can dep
   - Tu van dau tu
   - Xem phap ly
3. Ask if user is:
   - Dau tu sinh loi
   - Mua de o / nghi duong
4. Ask budget range
5. Collect:
   - Name
   - Phone/Zalo
   - Need
6. Push action:
   - Goi nhanh 2 phut
   - Gui Zalo truoc
   - Dat lich xem du an

UI requirements:
- Floating chat bubble bottom right
- Auto popup after 6 seconds
- Modern luxury style
- Mobile first
- Save all leads to CRM / Google Sheet / Firebase

Tone:
- Professional
- Fast
- Persuasive
- Not spammy
